# Changelog

All notable changes to CrispyUI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-23

### Added
- Initial release of CrispyUI
- Core addon framework with event-driven architecture
- Frame Styling module with modern blue theme
  - Custom backdrops for all major UI frames
  - Signature crispy blue borders (RGB: 51, 153, 255)
  - Subtle glow effects on frames
  - Styled close buttons
- Action Bar enhancements
  - Enhanced button borders and icons
  - Smooth hover animations with blue glow
  - Custom hotkey styling
  - Improved cooldown display
  - Support for all default action bars
- Unit Frame improvements
  - Larger health bar option (20% increase)
  - Class-colored health bars
  - Portrait border enhancements
  - Status bar glow effects
  - Enhanced visual contrast
- Tooltip enhancements
  - Automatic item level display
  - Optional item ID display
  - Modern backdrop styling
  - Configurable font size
  - Visual separators
- Chat Frame modernization
  - Modern clean appearance
  - Enhanced font readability
  - Styled edit boxes with focus effects
  - Configurable auto-fade (10s default)
  - Modernized buttons
- Configuration system
  - SavedVariables support for persistent settings
  - Per-module enable/disable toggles
  - Color customization options
  - Runtime configuration
- Slash commands
  - `/cui` or `/crispyui` for help
  - `/cui toggle` to enable/disable
  - `/cui status` for current status
  - `/cui reload` for UI reload
- Documentation
  - Comprehensive README
  - Detailed installation guide
  - Feature documentation
  - GPL v3 license

### Technical Details
- Interface version: 110002 (The War Within)
- No external dependencies (standalone)
- Modular architecture for easy maintenance
- Efficient event handling
- Minimal performance impact

### Notes
- Created as an experimental AI voice-coding workflow project
- Purely cosmetic enhancements, no gameplay modifications
- Compatible with most other add-ons
- Settings saved per account in SavedVariables
