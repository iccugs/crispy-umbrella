# CrispyUI - World of Warcraft Interface Add-on

A cosmetic overhaul for the World of Warcraft interface, providing modern styling and visual enhancements.

**Note:** This add-on was created as an experiment using AI-assisted voice coding - the repository contains the finished add-on product, not the workflow itself.

## Features

- **Modern Frame Styling** - Sleek, consistent visual theme across all UI frames
- **Enhanced Action Bars** - Improved button styling with hover effects and animations
- **Beautiful Unit Frames** - Class-colored health bars and enhanced portraits
- **Smart Tooltips** - Additional information display with item levels and IDs
- **Styled Chat Frames** - Modern chat appearance with customizable fade times

## Installation

1. Download the latest release
2. Extract the `crispy-umbrella` folder to your `World of Warcraft/_retail_/Interface/AddOns/` directory
3. Rename the folder to `CrispyUI` (optional but recommended)
4. Restart World of Warcraft or reload UI (`/reload`)

## Usage

### Slash Commands

- `/cui` or `/crispyui` - Show available commands
- `/cui toggle` - Enable/disable the add-on
- `/cui status` - Show current status
- `/cui reload` - Reload the UI

## Customization

The add-on stores settings in SavedVariables. Edit `Config.lua` or use in-game commands to customize:

- Frame border colors and sizes
- Action bar spacing and visibility
- Unit frame enhancements
- Tooltip information display
- Chat frame styling and fade timers
